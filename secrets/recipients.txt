# recipients.txt — the public keys allowed to decrypt this vault.
#
# One SSH public key per line, in the usual `ssh-ed25519 AAAA… comment` form
# (the same line you'd find in ~/.ssh/id_ed25519.pub). Every key listed here is
# passed to age's `-R` flag at seal time, so every listed participant — and only
# they — can later decrypt the sealed *.age files with their private key.
#
# This file is committed: public keys are not secret. To add or remove someone,
# edit this list, then unseal + re-seal so the blob is re-encrypted to the new
# set (see the README).
#
# Replace the placeholder below with your own key:
#   cat ~/.ssh/id_ed25519.pub
#
# ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAA...replace-me... you@example.com
